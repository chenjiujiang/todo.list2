# 待办事项清单 (Todo List)

一个简单而实用的待办事项管理网页应用。

## 功能特点

- 添加新的待办事项
- 标记待办事项为已完成/未完成
- 删除待办事项
- 过滤显示（全部/未完成/已完成）
- 清除所有已完成的待办事项
- 数据本地存储，刷新页面后数据不会丢失

## 使用方法

1. 在输入框中输入待办事项内容
2. 点击"添加"按钮或按回车键添加
3. 使用复选框标记完成状态
4. 使用过滤按钮查看不同状态的待办事项
5. 使用删除按钮删除单个待办事项
6. 使用"清除已完成"按钮批量删除已完成的待办事项

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- LocalStorage API
